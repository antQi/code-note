# 排序算法
## O(n^2)
* bubbleSort－冒泡排序 
  |  数量  |     用时   |
  |-------|------------|
  |100    | 0.415ms    |
  |1000   | 7.805ms    |
  |10000  | 763.115ms  |
  |10000  | 71891.106ms|

* selectionSort－选择排序 
  |  数量   |    用时     |
  |--------|-------------|
  | 100    | 0.240ms     |
  | 1000   | 2.230ms     |
  | 10000  | 105.640ms   |
  | 100000 | 6739.567ms  |

* insertionSort－插入排序  
  `接近有序的大量数据时会更快`
  |  数量  |     用时   |
  |-------|------------|
  |100    | 1.782ms    |
  |1000   | 2.816ms    |
  |10000  | 33.037     |
  |100000 | 2740.663ms |
       
## O(n*log n)
* mergeSort－归并排序   
  `时间上更快了，但是空间上多了`
  |  数量   |    用时     |
  |--------|-------------|
  | 100    | 0.240ms     |
  | 1000   | 2.230ms     |
  | 10000  | 105.640ms   |
  | 100000 | 6739.567ms  |
