# 资源合并与压缩-HTTP 请求的过程与潜在的性能优化点

## 目标

* 减少 HTTP 请求数量-合并
* 减少请求资源大小-压缩

##  怎么做

现代化的前端基本采用两种方式：

1、  前端构建工具

2、 后端模版引擎

* HTML 压缩
  * 压缩没有意义的字符 ，包括注释
    * 构建阶段压缩
    * 后端模版引擎
* CSS 压缩
  * 无效删除
  * CSS 语义合并
* Js 压缩与混乱
  * 删除无效字符
  * 删除注释
  * 代码语义的缩减和优化
  * 代码保护（混淆语义）
* 启用 gzip
