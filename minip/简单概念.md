# 小程序

## 小程序文件类型

- 样式 wxss-css
- 骨架 wxml-html (小程序组件)
- 业务 js-js
- 配置 json

## 基本结构

小程序

- app.json (全局配置)
- app.wxss (全局 css)
- app.js (全局 js)
- Page
  - wxml
  - wxss
  - js
  - json

## 良好的结构（组件式编程）

小程序

- app.json (全局配置)
- app.wxss (全局 css)
- app.js (全局 js)
- Page
  - wxml
  - wxss
  - js
  - json
  - component1

    - wxss
    - js
